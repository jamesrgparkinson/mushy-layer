Detailed Description

Nonlinear variable coefficient operator.

Operator for solving variable-coefficient (alpha * aCoef(x) * I - beta * Div(bCoef(x) . Grad)) phi = rho over an AMR hierarchy.

int AMRNonLinearVCOp::m_primaryVar

The primary variable for this solver.

Solves depend on both enthalpy and bulk concentration, but we only solve for one at a time. This variable specifies which, whilst the other is lagged in calculations.
